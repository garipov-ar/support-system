# Система технической поддержки с Telegram-ботом

## Описание проекта

Данный проект представляет собой прототип системы технической поддержки,
предназначенной для хранения и предоставления технической документации
(инструкций, прошивок, конфигураций, прайс-листов) через Telegram-бота.

Система разработана в рамках выпускной квалификационной работы (ВКР)
и реализует основной функционал доступа к документации через удобный
мессенджерный интерфейс.

---

## Цель работы

Разработка серверной системы и Telegram-бота для централизованного
хранения и распространения технической документации оборудования
различных производителей.

---

## Основные функции

- управление категориями и документами через веб-интерфейс администратора;
- иерархическая структура документации;
- REST API для доступа к данным;
- Telegram-бот для навигации по структуре документации;
- получение файлов пользователями через Telegram.

---

## Архитектура системы

Система реализована по клиент-серверной архитектуре и включает следующие компоненты:

- сервер приложений на базе Django;
- база данных PostgreSQL;
- REST API для доступа к данным;
- Telegram-бот, взаимодействующий с API;
- контейнеризация с использованием Docker.

---

## Используемые технологии

- Python 3.11
- Django
- Django REST Framework
- PostgreSQL
- Telegram Bot API
- Docker / Docker Compose

---

## Запуск проекта

Для запуска проекта требуется установленный Docker и Docker Compose.

```bash
docker-compose up -d --build
```
